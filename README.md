# Cleonara Fashion — Link Page

Halaman link (linktree-style) untuk Cleonara Fashion — grosir & ecer tas wanita.

## Isi halaman
- Chat Admin WhatsApp
- Shopee (Grosir & Reguler)
- Instagram
- Grup Info Harga
- Channel WhatsApp

## Cara host gratis via GitHub Pages
1. Push repo ini ke GitHub (lihat langkah di chat).
2. Di repo GitHub, buka **Settings > Pages**.
3. Pada **Branch**, pilih `main` dan folder `/ (root)`, lalu **Save**.
4. Tunggu 1–2 menit, link publik akan muncul di halaman yang sama, formatnya:
   `https://<username-github-kamu>.github.io/<nama-repo>/`

## Struktur
```
index.html   -> halaman utama (logo sudah tersemat langsung di file ini)
```
